基于随机森林的学生成绩多因素预测模型


本项目通过分析学生习惯数据，构建学业成绩预测模型，识别影响成绩的关键因素，并为低表现学生提供干预建议。项目涵盖数据预处理、特征工程、模型训练及可解释性分析全流程。
